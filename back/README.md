# API for project Arena

Prerequisites:
- Node.js v12.16 and NPM
- a Postgre DB
- A .env file in this folder folder with the following information:
```
# .env
PGHOST=
PGPORT=
PGDATABASE=
PGUSER=
PGPASSWORD=
```

## Project setup
```
npm start
```

### Hot-reloads for development (using nodemon)
```
npm dev
```
